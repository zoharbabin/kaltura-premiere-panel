# Plugin Icons

All 4 required icon sizes are present for Adobe Exchange distribution:

| File           | Size    | Usage                      | Source              |
| -------------- | ------- | -------------------------- | ------------------- |
| `icon-24.png`  | 24x24   | Panel title bar            | Kaltura Sun (color) |
| `icon-48.png`  | 48x48   | Plugin manager             | Kaltura Sun (color) |
| `icon-96.png`  | 96x96   | Exchange listing thumbnail | Kaltura Sun (color) |
| `icon-192.png` | 192x192 | Exchange hero image        | Kaltura Sun (color) |

The SVG source (`kaltura-sun-color.svg`) is also included for use in the login panel.

PNGs were generated from the official Kaltura Sun SVG using `rsvg-convert`.
